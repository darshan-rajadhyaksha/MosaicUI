| Prop          | Type                                                                 | Required | Default  | Description |
|---------------|----------------------------------------------------------------------|----------|----------|-------------|
| text          | `string`                                                             | No       | —        | Simple text to type. Use for basic usage instead of `words`. |
| words | `Array<{ text: string; speed?: number; slotProps?: object }>` | No       | —        | Advanced configuration for typing multiple entries. `text` defines the actual content to display, `speed` lets you control typing speed per word (overriding the global `speed`), and `slotProps` allows custom styling or additional props (e.g., color, className) for that specific word—giving you fine-grained visual and behavioral control. || hideCursor    | `boolean`                                                            | No       | `false`  | Hides the typing cursor when set to `true`. |
| blinkCursor   | `boolean`                                                            | No       | `true`   | Enables cursor blinking animation. |
| cursorVariant | `"line" \| "block" \| "underscore"`                                  | No       | `"line"` | Controls the visual style of the cursor. |
| speed         | `number`                                                             | No       | `100`    | Default typing speed in milliseconds (used if not overridden in `words`). |
