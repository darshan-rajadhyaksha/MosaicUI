| Prop   | Type        | Required | Default | Description |
|--------|------------|----------|---------|-------------|
| count  | `number`     | No       | `6`       | Number of bars in the wave animation (minimum 5). If greater than 6, custom colors must be provided. |
| colors | `string[]`   | No       | `[ "#eade3d", "#2eeca8", "#ec902e", "#09b7bf", "#55acee", "#ea3d8c" ]` | Array of colors applied to bars. If `count` is more than 6 and colors are not provided, the loader falls back to the current text color. |
