Components

Explore all available components and find the ones you need for your project.

Circular Text

A flexible component for rendering text along a circular path with customizable styling and rotation.

Classic Accordion

A classic accordion component that expands and collapses content sections to organize information efficiently.

Colorized Text

Applies customizable multi-color styling to text content.

Decrypting Text

Displays text with a decrypting animation effect, revealing the final content through randomized characters.

Dot Grid Gradient

A customizable dot grid with gradient scaling, allowing control over dot size, color, and shrink direction.

Falling Tiles

Tiles fall from the top, settle into a grid for a moment, and then descend one by one in a looping animation.

Floating Indicator Tabs

A lightweight accessible tab component designed with a fluid motion indicator that delivers smooth, spring-based transitions.

Ghost Cubes

Translucent cubes drift upward and fade away, creating a subtle ghost-like background motion.

Grid Pattern

A customizable animated grid background component for creating dynamic patterned layouts behind content.

Image Comparison Slider

An interactive before-and-after image comparison slider that lets users visually explore differences with drag control.

Maze Pattern

A maze-themed background with customizable colors and size.

Modular Grid

A composable layout grid that provides responsive breakpoints and configurable variants for building structured UI sections.

Night Sky

A dynamic night sky background with twinkling stars, customizable density, and content layered on top.

Orbit Toggle Switch

A dynamic toggle switch where both the track and thumb rotate smoothly when switching states, creating a visually engaging interaction.

Rolling Letters

Displays a word where each character animates with a vertical rolling effect, creating a dynamic text reveal.

Shining Text

A smooth light shine moves across the text, making it look bright and eye-catching.

Star Field

A dynamic star field background with adjustable speed, creating a sense of motion and depth.

Text Emerge

Animates text by gradually turning blurry letters into clear ones, word or letter by letter.

Text Reveal

A text with a moving blob that reveals the text as it passes.

Three Dots

A minimal three-dot animated loader that indicates ongoing processing or content loading.

Typewriter Animation

Animates text like a typewriter, with optional speed, cursor, and styling controls.

Vertical Text Slider

Slides through an list of text vertically, pausing briefly on each item before transitioning to the next.

Wave Bars

A rhythmic wave loader with cascading vertical bar motion.
